class PyEvoMotion.core.parser.PyEvoMotionParser(input_fasta: str, input_meta: str, filters: dict[str, list[str] | str], positions: tuple[int], date_range: tuple[datetime] | None)[source]

Bases: object

This class is responsible for parsing the input fasta and metadata files. It is inherited by the PyEvoMotion class.

Parameters:

  • input_fasta (str) – The path to the input .fasta file.

  • input_meta (str) – The path to the input metadata file. The metadata file must contain a date column and can be in either .csv or .tsv format.

  • filters (dict[str, list[str] | str]) – The filters to be applied to the data. The keys are the column names and the values are the values to be filtered.

  • positions (tuple[int]) – The start and end positions to filter the data by.

  • date_range (tuple[datetime] | None) – The start and end dates to filter the data by. If None, the date range is not filtered.

On construction, it invokes the following methods:

  • parse_metadata(): Parses the metadata file.

  • parse_sequence_by_id(): Parses the sequence with the ID of the first entry in the metadata file.

  • filter_columns(): Filters the metadata based on the input filters.

  • filter_by_daterange(): Filters the metadata based on the input date range.

  • parse_data(): Parses the input fasta file and appends the mutations that differ between the reference sequence and the input sequences to the metadata.

  • filter_by_position(): Filters the metadata based on the input start and end positions.

Attributes:

data: pd.DataFrame

DataFrame containing metadata.

reference: SeqRecord

The reference sequence parsed from the fasta file.

Module contents

The main functionality of the PyEvoMotion project is abstracted into the following classes:

  • PyEvoMotion - The main class that encapsulates the entire analysis.

  • PyEvoMotionBase - The base class that provides basic utility functions inherited by PyEvoMotion.

  • PyEvoMotionParser - The class that provides the functionality to parse the input data for the analysis, inherited by PyEvoMotion.
